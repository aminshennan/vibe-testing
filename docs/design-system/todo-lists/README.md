- **User-Centered Design** - Design decisions based on user needs and behavior
- **Consistency** - Unified visual language across all pages
- **Visual Hierarchy** - Clear information architecture and content prioritization
- **Typography** - Readable, accessible, and purposeful text styling
- **Whitespace (Negative Space)** - Strategic use of space for clarity and focus
- **Alignment** - Precise positioning and visual order
- **Grid Systems** - Structured, responsive layout foundations
- **Color Theory & Contrast** - WCAG-compliant color usage for accessibility
- **Balance** - Visual weight distribution for harmony

- **Repetition** - Consistent patterns that create unity

- **Clarity** - Clear communication and easy comprehension
- **Familiarity** - Use of known patterns and conventions
- **Scalability** - Design works as content grows
- **Modularity / Reusability** - Component-based design system
- **Performance-aware design** - Optimized for speed and responsiveness
- **Aesthetic-Usability Effect** - Beautiful design enhances perceived usability
